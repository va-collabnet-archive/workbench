1. Classifying the terminology    

The IHTSDO Workbench IDE contains a high performance classifier (the SnoRocket Classifier) for validating the inferred relationships inherent in the terminology, according to the Description Logics upon which SNOMED CT is based. (1). The SnoRocket Classifier is a highly optimised module for performing complete or incremental classification of SNOMED CT and which can be configured through business process plugins in a number of ways. Additional information on configuration and performance characteristics for developers can be found in Integrating the SnoRocket Classifier.
In most instances, the Workbench IDE is bundled with a Classifier plugin, accessable via the main menu. The following sections illustrate its use.
