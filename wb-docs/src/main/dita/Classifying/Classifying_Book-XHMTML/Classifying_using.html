1.1. Using the Classifier    

The classifier is executed through the business process plugin and can take several minutes. Once the classifier has been run, the Normal Form results for a given concept can be shown in the component panel, by selecting the toggle. Ensure that the STATE_INFERRED viewing option is checked in the Component Pane of the Preferences Panel.
The SnoRocket Classifier is a highly optimised module for performing complete or incremental classification of SNOMED CT. The IHTSDO Workbench provides support for using this classifier via in-built tasks available from within the Business Process Builder. The classifier related tasks available can be seen in Figure 1
