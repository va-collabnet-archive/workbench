1.2. Setting up for Classifying    

About this task

In many cases, the IHTSDO Workbench will have the Classifier setup already in-place as "Run Classifier" under the Tasks menu. Where installations don't have access to the classifier plugin via the main menu, here is a brief outline of the steps needed to set up (or, to review settings for) the IHTSDO Workbench to run the built-in classifier against editing changes that have been made to a terminology:

Procedure

  1. Set up the "Edit Path" using the Preferences Panel as outlined in Setting Classifier Preferences.
    Use the "New Path" tab to establish the new edit path:-

    Figure 2. create a new edit path

  2. Create a "Classify Path" using the Preferences Panel.
    Return to the "New Path" of the Preferences Panel, create another path for the classifier...

    Figure 3. create a new classifier path

    Review the path settings under the Preferences "Path" Tab. Only the "Edit Path" should have "Edit on this path" checked. Both the "Edit Path" and "Classifier Path" should have the "Use position in view" checked.
    The resulting paths should look similar to the following in the Terminology Auxiliary hierarchy in the taxonomy panel ...

  3. Set up the 4 Classifier preferences using the Classifier Tab of the Preferences Pane
    Navigating to the appropriate terms in the taxonomy view, drag and drop the following concepts to their respective targets in the classifier preference pane.

    Table 1. Drag and Drop editing of classifier preferences

    Source Concept Classifier Parameter
    SNOMED CT Concept Classification root
    SNOMED CT Concept / Linkage Concept / Attribute / Is a Classification is-a
    SNOMED CT Concept / Linkage Concept / Attribute / Concept model attribute Role root
    Terminology Auxiliary concept / path / development / Edit Path Classification Input Path
    Terminology Auxiliary concept / path / development / Classifier Path Classification Output Path
    The resulting 5 classification parameters should resemble the following figure ...

  4. Create a simple "Classifier" Business Process using the Business Process Builder
    1. Navigate to the IDE -> Classify category in the task navigator pane

    2. and drag the "Classifier (Current)" onto the process builder pane
  5. Store the Classifier Business Process as a menu item by saving in a subfolder of the "plugins/menu" folder.
