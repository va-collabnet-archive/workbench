Indexing translation memory

When selecing this option, a window will be automatically displayed in the lower third of the screen.

(Note: click if the working window is not automatically displayed.

  • Press the bar displayed on the bottom of this window:

A window is opened asking you to select the file that will be indexed. It should be an Excel file with two columns: Source and Target, where descriptions in the source and target languages are entered.

Excel 2007 documents ( . xlsx) are recognized, as well as .xls files.

Upon clicking the ‘Open file’ button, each row of the table is entered as a line. (An error message will be displayed in case the process cannot be completed.)

The translator will see any of these whenever he/she selects a concept that contains any of the terms included in the Translation Memory in the middle window of the Translation help tab:
