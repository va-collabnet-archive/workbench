ICS generation strategy

  • Specifies how will the Initial Capital Status will be managed, as there are words that should always be written with uppercase (e.g.,Turner syndrome) or lower case (e.g., pH) regardless the editorial guidelines related to the capitalization status of the translated descriptions:
    • None: ICS is false by default. This means that the initial capital status is not significant.
    • Copy from source: defines the default ICS as the same of the source FSN
    • Words list: uses a list of case significant words for defining the default ICS. This feature is not yet implemented in this version
