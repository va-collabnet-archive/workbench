Index dictionary

In order to make searches in dictionaries or activate the spell checker for your translations, dictionaries must be indexed first.

The Index dictionary tool allows indexing dictionaries from different sources. Once you have selected Index Dictionary option, the following window is opened in the lower part of the screen:

When “Index dictionary from text file” button is selected, an open file option is opened. Every workflow bundle has a Dictionaries folder. You must open it and select a language, that will be automatically indexed:

Once you have ended indexation you may open Search dictionary tool.
