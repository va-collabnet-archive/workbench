Project left panel

The left panel displays the different parts of your translation project in a tree view, each of them identified by a distinct icon. Clicking on each icon will display the corresponding details pane on the right.
  • Select “Open Project Manager” from the upper left menu. The project manager tab is displayed as a Translation Projects folder.
  • Double clicking will display the translation project tree.
  • Click the ‘Add project’ button to create a new project:

  • A new window is displayed requesting to enter a new project name. Enter a new project name and click "OK":

  • The new project will be displayed under the Translation Projects folder on the left.

  • Now you must define its main characteristics. To do so, click your prject's name and go to the details panel on the right.
